# MTG Card Scraper

Search for Magic the Gathering cards at local game shops, compare prices, and more. Currently supports any game shop with a Crystal Commerce website.

## Dependencies

- [fzf](https://github.com/junegunn/fzf) - for any script that searches for cards
- `requests` - for downloading pages
- `beautifulsoup4` - for scraping the webpages

## Installation

```
$ pip install requests beautifulsoup4
$ brew install fzf
```
